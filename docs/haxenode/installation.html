Installation

Installing

Okay I don't want to mention this again, but you need to install Node.js and Haxe first (read more about that here)

Install Node externs for Haxe

The HaxeFoundation maintains the hxnodejs externs.

Beside the node externs I suggest you install https://github.com/clemos/haxe-js-kit as well. It has all the externs this documentation based upon.

Install hxnodejs

Install hxnodejs via haxelib:

haxelib install hxnodejs

Install js-kit using haxelib:

You install via git:

haxelib git js-kit https://github.com/clemos/haxe-js-kit.git haxelib

Optional, check that js-kit installed correctly:

haxelib list

Search for js-kit:

...
js-kit: git [dev:/usr/lib/haxe/lib/js-kit/git]
...
